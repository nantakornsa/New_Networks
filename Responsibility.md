# 👥 Roles, Responsibilities & Boundaries

---

## Team Members

| Name | Role |
|------|------|
| ชินวัตร แสนเมือง | Project Architect |
| นันทกร แสวงจิตร | Network Engineer |
| เกียรติพงษ์ เผดิมศักดิ์ | AI Specialist |
| ภูริ ตั้งพงษ์ | Security & Ethics Analyst |
| ฉัตรนัย ไกรราช | Scrum Lead |

---

## Role Responsibilities

### Project Architect
- ออกแบบ Architecture
- กำหนด NCP
- ตัดสินใจด้านเทคโนโลยีหลัก

### Network Engineer
- ออกแบบ Neural Router
- วิเคราะห์ Routing Protocol
- Simulation Network Flow

### AI Specialist
- ออกแบบ ASD
- Intent Recognition
- Emotion Encoding Model

### Security Analyst
- ออกแบบ Neural Firewall
- วิเคราะห์ Privacy Risk
- Data Protection Framework

### Scrum Lead
- จัดทำ Sprint Plan
- ควบคุม Timeline
- รวมเอกสารส่ง

---

## Boundaries

- Architect ไม่ออกแบบ AI Model ลึก
- AI Specialist ไม่ยุ่ง Network Layer
- Security Analyst ให้คำแนะนำ แต่ไม่ override Architecture
- Scrum Lead ไม่ตัดสินใจเชิงเทคนิค
