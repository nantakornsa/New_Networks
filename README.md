# Neural Communication Network

## Project Overview
Neural Communication Network คือระบบสื่อสารผ่านชิปสมองที่เชื่อมต่อกับเครือข่ายดิจิทัล
ผู้ใช้สามารถส่งข้อความ โทรศัพท์ หรือแลกเปลี่ยนข้อมูลผ่านความคิดได้โดยตรง
